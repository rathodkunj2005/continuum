# FNDR iOS app scaffold

SwiftUI app shell for the FNDR companion.

Included surfaces:
- Ask (query + answer + source cards)
- Memories (search + detail)
- Capture (manual capture + offline queue fallback)
- Status (reachability + capture controls)
- Settings (pairing, cache mode, app-lock stub)
- Pairing (QR payload parse + pair complete)
- App Intents stubs (Ask, Remember, Pause)

This folder is wired into `apps/ios/FNDR.xcodeproj` via `apps/ios/project.yml`.
